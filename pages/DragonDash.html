Dragon Dash Game Development

This page is for my game Dragon Dash.

What will you see in this page? - You will see all features in this game as well as a video showcasing the game itself. I will also be giving information and more details on how the project was made and why.


Dragon Dash Video


Info on video
  • This video showcases the core gameplay loop, where the player goes throguht levels to beat the game. this game was made for an arcade machine, meaning that there are only 6 buttons max, and i had to take into consideration that the arcade used a joystick instead of a normal keyboard.


Wall Climbing

I decided to add this feature to make parkour or secret areas harder to spot for the player. Thus adding new things to explore and hidden mini bosses and loot to find.

Boss Battle

As an extra feature i decided to add a boss battle to my short arcade game, where the player has to avoid the huge ice golem at the 2nd level.



Player Attack
Shooting

The player has an attack for shooting fireballs out of his mouth, meaning he has to be grounded when shooting.

Enemies
Melee & Ranged

Ranged and melee enemies use an easy to dodge attack that only activates when a certain distance.

Score
Highscore System

I added a highscore board for the player to try and get the highest score he can.



Health System

The health system is based on hearts, and you can replenish each heart by collecting healing items in the level.

Controls Pannel

I had made a controls screen in the main menu so that it would be easier for someone to start playing the game on the arcade.

Victory/Defeat Canvas

This was the screen which would pop up whenever you beat a level or died in the game, it could probably have been improved upon.



Notice
Class Assignment

This project was a class assignment and was meant to be used for the arcade and thus has a simple key layout.
