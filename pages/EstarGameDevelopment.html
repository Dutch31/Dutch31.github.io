Estar Game Jam

Development

Estar Game Development

Estar is a game that was made in a mini game jam. the theme was "Those shouldnt be mixed." and we decided to make an alchemy game in the 2 days we had.


Estar Game Video


Info on video
  • This video shows the core gameplay loop that will be used when further developing the game. But as you can also tell from the video the game was made in unity using C#, 4 people worked on it in total and we all worked hard and had a good time working together.


Crafting

Crafting is a big part of the game so we decided to focus our resources on it as one of the core features of our game.

Player

The player character was easily made as we just used an easy 2D movement script to control the character and make it so that we can interact with all objects in the room.



Objective
End goal of game

The goal is to not die and create the philosiphers stone.

Obstacles
The Difficulty

The main difficulty in this game is actually finding out what can be crafted and trying to not die by making the wrong things.

Game Time
Game Jam

This project was made in a mini game jam that lasted a little over a day, so around 28 hours or so.



Credits Scene Design

Designing the credits wasnt to difficult as i had found a good UI pack and was able to use it.

Crafting System

The crafting system was complex and we had in total around 40-50 ingrediants and a total of 40 recipies.

Start Menu

The start menu was not that difficult to create, but i do like the end product as it gives of a good vibe of entering an alchemists house.
