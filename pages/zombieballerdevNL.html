Zombie Baller Game Development

This was a group project made in a total of 8 weeks, where we had to brain storm, paper prototype, create assets, and program an arcade game. the game made out to be a top down zoombie shooter where the main goal is to stay alive as long as possible.


Zombie Baller Video


Info on video
  • In this video it shows how the game functions and how it works. The main mechanics and features are shown in this video as well as all art that was created by our digital artists in our group.


Speed Boost

The speed boost is an activation tile that gives you a boost in movement speed for a few seconds before turning of once more. These speed boosts can be found around the map.

Highscore Board

One of the objectives we had whilst creating this project was to make a highscore board, and we did not finish creating it so it barely work. If we had more time to work on the project we would have improved that first and foremost.



Enemy
Enemy AI

The enemy AI was made so that it would always follow the player and build up over time.

Controls
Arcade Controls

The arcade controls were using a joy stick so we made it where the player turns and looks at the same direction.

Player Attack
Shooting

The shooting mechanic was made used to kill the zombies, fire rate and damage was something we wanted to control for if we added more guns.



Health Pack

Health packs were found around the map for the player to find, so that we could heal and keep fighting the hordes of zombies.

Map

The map was made so that we could customize the streets and debree to our liking.

Menu

The menu was made so it was simple to navigate with the arcade joystick and fealt easy to use.
