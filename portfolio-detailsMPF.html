Magic Platformer Page

This is a platformer game where the player is a magician and has to go throught levels to get more powerfull and reach the final boss.


Spell Unlocks

Around all the levels in the game there will be spells books, hidden away in portals to different dungeons. where players can unlock spells and new ways to combat enemies.

Lighting

I used this project as a way of improving my skills and trying new things, so for the first time ever i used 2D lighting in my project. After carefull consideration i think it looks good and is a great way of improving a scene.



Enemy AI

The Enemy AI for the simple troops is similar to that of a goomba from mario, it walks from left to right at a certain area where it guards the root from the player. When the player is in sight it attacks and deals damage when possible.

Shop

The shop is where you can select and upgrade your spells and stats with coins found in each level.

Level Selector

The level selector can be found in the hub where you can select which level and the difficulty you want to challenge.

Layout

The layout of level 1 is simple, a straight path with 3 secret spell dungeons to set up basic combat. each dungeon is hidden and you have to walk into the area to get teleported.



Magic System Video

The video to my right showcases my first ever spell select system. where the player has to select which spell to use before going into combat or whilst the player is in the pause menu. I believe that this is a good system that can obviously be improved upon and made more eccessible. But for now it gets the job done and im proud of making it.
